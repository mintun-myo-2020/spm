## Question 1
Where should class notes live — per session (each class meeting gets its own notes) or per class (one running set of notes for the whole class)?

A) Per session — each class session gets its own notes (topic covered that day, homework given, weaknesses observed)
B) Per class — one set of notes for the whole class that the teacher updates over time
C) Both — per-session notes AND a general class-level description/notes field
D) Other (please describe after [Answer]: tag below)

[Answer]: A

## Question 2
What fields should class notes include?

A) Free-form text only — one big text area for the teacher to write whatever they want
B) Structured fields — separate fields for "Topic Covered", "Homework Given", "Common Weaknesses", plus an optional free-form "Additional Notes"
C) Structured fields with tags — same as B but with taggable topics (linked to the subject's topic list)
D) Other (please describe after [Answer]: tag below)

[Answer]: B

## Question 3

A) Teacher only — private notes for the teacher's own reference
