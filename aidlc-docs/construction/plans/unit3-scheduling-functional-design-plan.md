# Unit 3: Class Scheduling & Attendance — Functional Design Plan

## Plan Overview
This plan covers the functional design for the Class Scheduling & Attendance feature, the largest unit in Sprint 2. It introduces 3 new database tables, a new `scheduling` module, session generation logic, attendance tracking, student RSVP, and multi-role frontend views.
